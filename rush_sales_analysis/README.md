# RUSH: Sales Analysis

**Author:** Logan Kaliba  
**Date:** August 10, 2025

## Overview

This project analyzes multi-year sales data for **RUSH**, a fictional athletic apparel retailer. Using Python and exploratory data analysis techniques, I evaluated product performance, geographic trends, and customer purchasing patterns to identify opportunities for growth across product categories and regions.

The analysis answers key business questions around top-performing categories, state-level sales patterns, and retailer-level purchasing behavior. A presentation accompanies the notebook to communicate insights and recommendations clearly to business stakeholders.
